A simulation using the `pygame` package of the gravitational interaction between several sheprical bodies of mass.

Click and drag the mouse to spawn new bodies. Press the space bar to reset the center of mass of the system to the center of the window.

## Getting started

In order to get started, install the package by running

```shell 
uv sync
```

Then run the simulation by executing

```shell
uv run simulation
```
